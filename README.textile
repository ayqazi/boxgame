README
------

A game that may have some boxes in it.

This project is a place for me to test out / play around with some
game development.


INSTALLATION
------------

On Ubuntu/Debian, you will need the SDL libraries:

libsdl1.2-dev libsdl-gfx1.2-dev libsdl-image1.2-dev
libsdl-mixer1.2-dev libsdl-net1.2-dev libsdl-ttf2.0-dev libsdl-sge-dev
